# AutoBeamer
A beamer interpreter for regular LaTeX documents.

## Roadmap

1. .TEX -> .TEX a LaTeX document converter, with automatic `frame` adds.
2. .TEX -> .PDF attempt to modify beamer engine
3. .TEX -> .PDF sense the centralized idea by ML
