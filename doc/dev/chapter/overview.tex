\textsf{AutoBeamer} 实用程序旨在转换普通的 \LaTeXe{} 讲义文件翻译为可以编译为 Beamer 幻灯片的对应文档。

该程序尚处于开发初期。决定采用 Rust 语言编写该实用程序。
